Workforce Participation Visualizer – Gender Equality Project Project Title

Workforce Participation Visualizer Using C++

🌍 Project Description

The Workforce Participation Visualizer is a C++ based mini-project created to promote awareness about Gender Equality in Employment. This program reads workforce demographic data from a CSV file and generates simple ASCII bar charts in the terminal to visually represent the participation of men and women in different sectors.

The project helps users easily understand the gender gap in workforce participation through colorful text-based visualization and percentage analysis.

It is a beginner-friendly data visualization project that combines:

📊 Data Analysis 📁 File Handling 📈 ASCII Visualization 💻 C++ Programming 🌍 Social Awareness (Gender Equality)

This project supports the idea of equal opportunities for all genders in workplaces and demonstrates how technology can be used to spread awareness through data.

✨ Features ✅ Reads workforce data from CSV file ✅ Displays male and female participation percentages ✅ Generates ASCII bar charts ✅ Colorful console output ✅ Calculates gender gap automatically ✅ Simple and beginner-friendly C++ code ✅ Easy to customize with new datasets
